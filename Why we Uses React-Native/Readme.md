🚀 Why do we use React Native?

👉 Simple answer:
We use React Native to build mobile apps faster and easier using one codebase

🧠 1. One Code for Android + iOS

👉 Normally:

Android → Java/Kotlin
iOS → Swift

❌ Two different apps → more time

👉 React Native:

One JavaScript code → works on both 📱

✅ Saves time + effort

⚡ 2. Faster Development

👉 You don’t need to learn:

Java
Swift

👉 Just learn:

JavaScript + React Native

✅ Perfect for beginners like you

💸 3. Cost Saving (Very Important)

👉 Companies love React Native because:

❌ No need for 2 teams
✅ One developer can build both apps

👉 That’s why high demand skill

🔄 4. Reusable Components

👉 Write once → reuse many times

Example:

<Text>Hello</Text>

👉 You can use it anywhere

📱 5. Real Native Apps (Not Website)

👉 Apps built with React Native:

Feel like real apps
Smooth performance
Use phone features (camera, GPS, etc.)

🌍 6. Used by Big Companies

👉 Real companies using it:

Facebook
Instagram
Uber Eats

👉 So skill = valuable 🔥

🧩 Simple Analogy

👉 React Native is like:

🍜 “One recipe → cook for both Android & iOS”

🎯 When Should We Use It?

👉 Use React Native when:

✅ You want to build mobile apps
✅ You want faster development
✅ You want cross-platform apps
✅ You are beginner (like you)
