Day 2 – Internet + JavaScript + Node.js

🌐 1. How the Internet Works (Very Basic)

💡 Simple Idea:

Internet = Computers talking to each other

🧠 Example:

Jab tum browser me likhte ho:

www.google.com

👉 Yeh steps hote hain:

Browser request bhejta hai (like asking: “Google ka data do”)
Server (Google ka computer) response bhejta hai
Browser usko screen par show karta hai.

🔁 Real Life Example:
Tum → Customer
Internet → Road
Server → Restaurant
Website → Food 🍔

👉 Tum order dete ho → food aata hai

📌 Important Words:
Client = Tumhara mobile / laptop
Server = Jahan data stored hota hai
Request = Data maangna
Response = Data milna

🧪 Practice:

👉 Try karo:

Chrome open karo
Inspect (Right click → Inspect)
Network tab open karo
Refresh karo

👉 Tumhe requests dikhenge 🔥

2. What is JavaScript & Why Needed
   💡 Simple Definition:
   👉 JavaScript = Website ko “alive” banata hai

   🧠 Example:

Without JS:

Button click → kuch nahi hota ❌

With JS:

Button click → alert / login / animation ✅

🔥 Real Use:
Form validation (wrong password detect)
Button click actions
Animations
API data fetch (Instagram feed, etc.)

📌 Why Learn JavaScript (2026):
React Native (mobile apps) → JS se banta hai
Web + App dono → same language
Jobs me high demand 💰

🟢 3. What is Node.js (Simple + Job Understanding)

💡 Simple Definition:

👉 Node.js = JavaScript ko browser ke bahar run karne deta hai

🧠 Example:

Normally:

JS → sirf browser me run hota hai

Node.js ke saath:

JS → server bhi bana sakta hai

🔥 Real Use:
Backend banana (API)
Database connect karna
App ka server banana

📌 Why Important for Job:
Full stack developer ban sakte ho
React Native apps ke backend me use hota hai
Companies Node.js demand karti hain
